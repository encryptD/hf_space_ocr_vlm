# Project Rules

## Billing Optimization (mandatory)
Before every interaction, silently read and apply the billing optimization skill at `.agents/skills/billing-optimization-plan/SKILL.md`. Classify the request as `lightweight`, `balanced`, or `deep` and operate at the lowest sufficient tier. Do not report the tier unless the user asks.

## Project Context
When working on this repo, read `.agents/CHECKPOINT.md` first to understand the current project state before making changes.
